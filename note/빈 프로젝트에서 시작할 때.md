# Vite + React + TypeScript 수동 설치 가이드

빈 폴더에서 Vite로 React 프로젝트를 수동 구성하고, 필수 환경(TypeScript, React Compiler, Tailwind CSS)을 설정하는 방법.

---

## 1. 프로젝트 생성 및 초기화

WebStorm 터미널 또는 시스템 터미널에서 아래 명령어 입력.

```bash
# 현재 폴더(.)에 Vite 프로젝트 생성
npm create vite@latest .
```

**프롬프트 선택지 가이드**

- Current directory is not empty → **Remove existing files and continue** 선택 (가장 깔끔함)
- Select a framework → **React**
- Select a variant → **TypeScript + React Compiler** (최신 최적화 도구 포함)

---

## 2. 필수 패키지 설치

프로젝트 구조가 생성되면 필요한 라이브러리 설치.

```bash
# 기본 의존성 설치
npm install

# Tailwind CSS 및 관련 도구 설치
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

---

## 3. 핵심 설정 파일 구성

### tailwind.config.ts (확장자 변경 권장)

`tailwind.config.js`를 `.ts`로 변경하고 프로젝트 색상 토큰을 정의.

```typescript
import type { Config } from 'tailwindcss'

export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {
      colors: {
        navy: {
          DEFAULT: '#002147',
          light: '#1A3051',
          dark: '#343F5B',
        },
        red: '#FF2E4D',
        gold: '#FFB800',
        green: '#00FFAB',
      },
    },
  },
  plugins: [],
} satisfies Config
```

### postcss.config.js

기본 설정 유지. (JS 확장자로 두어도 무방.)

```javascript
export default {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}
```

---

## 4. 실행 및 확인

```bash
npm run dev
```

브라우저에서 `localhost:5173` 접속 후 화면이 정상적으로 나오는지 확인.
